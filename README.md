## Requirements
- Python 3.8
- PyTorch 1.10.0
- torchvision 0.11.0
- Cuda 11.3
- OpenCV
- Numpy
- tqdm
- ninja

You need to compile the ops provided by [stylegan2-pytorch](https://github.com/rosinality/stylegan2-pytorch) using ninja:

```
cd third_libs/stylegan_ops
python3 setup.py install
```
